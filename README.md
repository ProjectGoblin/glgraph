# ros-graph

This package is a dependency for [globin][goblin], providing JS/CoffeeScript
alternative for the official Python package `rosgraph`

## Status

Following components are completed and fully tested by the same (or more strict)
unit test cases used by the official Python package.

+ rosgraph.names
+ rosgraph.rosenv

[goblin]: https://github.com/ProjectGoblin/goblin
